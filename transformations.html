3.12. Geographic coordinate transformations¶

Perform various coordinate transformations

This module performs various coordinate transformations, based on some well-known formulas. The rationale for this module is that gdal is non-intuitive to use for the needed transformations, but more importantly, is very hard to install, especially on the Windows platform.

class sapphire.transformations.FromWGS84ToENUTransformation(ref_llacoordinates)¶
ecef_to_enu(coordinates)¶

Convert from ECEF coordinates to ENU coordinates

ENU: East, North, Up

The conversion formulas are taken from http://en.wikipedia.org/wiki/Geodetic_system#From_ECEF_to_ENU

Parameters:coordinates – a tuple containing the ECEF coordinates of the point to transform
geode = <sapphire.transformations.WGS84Datum object at 0x10771a4d0>¶
lla_to_ecef(coordinates)¶

Convert from LLA coordinates to ECEF coordinates

LLA: Latitude, Longitude, Altitude ECEF: Earth-Centered, Earth-Fixed

The conversion formulas are taken from http://en.wikipedia.org/wiki/Geodetic_system#From_geodetic_to_ECEF but slightly reworked.

Mind that the input is expected to be in degrees, as is standard in coordinate notation.
